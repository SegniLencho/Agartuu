Agarsiisa Tesla Cybertruck

Teeslaan dhaabbata Konkoolaata Electrikaan deeman oomishuun kan beekamu yoo ta'u kanaan dura konkoolaataa electrikaan deemaan oomishee gabaadhaaf kan dhiheesse yoo ta'u. barana ammoo konkolaataa guddaa kan Cybertruck jedhamu oomishee agarsiisaaf dhiheesseera. Konkolaataan kun gosa motoraa sadiin kan oomishame yoo ta'u. Kan Motora tokkoo(Single Motor), Kan Motora lamaa(Dual motors) fi kan Motora sadii (Tri Motors).

Saffiisni konlolaataa kanaa konkolaa sport (Sports Cars) waliin kan wal madaalu ta'ee. Ariitii Zeroo irraa ka'ee Kiloomeetira 97 gahuuf secondi 6 fi walakkaa qofa itti fudhata. Dabalataaniis Dandeettii Soolarii irraa humna fudhachuu ni qaba. Jabinni Qaamaa isaa yeroo konkolaatotta biroo waliin madaalamu baay'ee jabaataafi kan rasaasni keessa hin baane jedhamee yaadama.
